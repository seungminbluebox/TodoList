# 📘 ToDo List 사용자 가이드

이 문서는 **ToDo List** 애플리케이션의 사용 방법을 안내합니다. 이 앱을 통해 할 일을 체계적으로 관리하고, 일정을 확인하며, 생산성을 높일 수 있습니다.

## 1. 시작하기

웹 브라우저에서 `index.html` 파일을 열면 애플리케이션이 실행됩니다. 별도의 설치 과정 없이 바로 사용할 수 있습니다.

## 2. 주요 기능 사용법

### 📝 할 일 추가하기

1. **카테고리 선택**: 입력창 왼쪽의 드롭다운 메뉴에서 `개인`, `업무`, `공부`, `기타` 중 하나를 선택합니다.
2. **내용 입력**: 할 일 내용을 텍스트 입력창에 작성합니다.
3. **기한 설정 (선택)**: 달력 아이콘을 눌러 마감 기한을 설정할 수 있습니다.
4. **추가 버튼**: `추가` 버튼을 누르거나 엔터키를 입력하여 리스트에 등록합니다.

### ✅ 할 일 관리

- **완료 처리**: 리스트 항목 왼쪽의 체크박스(☑️)를 클릭하면 완료 상태로 변경되며, 취소선이 그어집니다. 완료된 항목은 하단의 '완료된 할 일 보기' 목록으로 이동합니다.
- **수정**:
  1. `수정` 버튼을 클릭합니다.
  2. 텍스트 입력창이 활성화되면 내용을 수정합니다.
  3. `저장` 버튼을 눌러 변경 사항을 반영합니다. (수정 중에는 드래그 앤 드롭이 제한됩니다.)
- **삭제**: `삭제` 버튼을 클릭하면 항목이 **휴지통**으로 이동합니다.

### 📅 캘린더 뷰

- **일정 확인**: 화면 왼쪽의 달력을 통해 날짜별 일정을 확인할 수 있습니다. 마감 기한이 있는 할 일이 존재하는 날짜에는 별도의 표시(점)가 나타납니다.
- **상세 보기**: 날짜를 클릭하면 해당 날짜가 마감인 할 일 목록이 달력 아래에 표시됩니다.
- **드래그 앤 드롭 기한 변경**: 할 일 목록에 있는 항목을 드래그하여 달력의 특정 날짜에 놓으면(Drop), 해당 날짜로 마감 기한이 즉시 변경됩니다.
- **스마트 스크롤**: 캘린더와 메모 영역은 화면을 스크롤해도 항상 보이는 위치에 고정(Sticky)되어 편리하게 사용할 수 있습니다.

### 🗑 휴지통 및 복구

- **휴지통 보기**: 화면 하단의 `🗑 휴지통 보기` 버튼을 클릭하여 삭제된 항목을 확인합니다.
- **복구**: 실수로 삭제한 항목은 `복구` 버튼을 눌러 다시 할 일 목록으로 되돌릴 수 있습니다.
- **영구 삭제**: `영구 삭제` 버튼을 누르면 데이터가 완전히 사라지며 복구할 수 없습니다.

### 📝 메모 기능

- 할 일 목록에서 특정 항목의 텍스트를 클릭하면 우측 **메모** 영역이 활성화됩니다.
- 해당 할 일에 대한 상세한 메모를 작성하고 `저장`할 수 있습니다.
- 메모 영역 또한 스크롤 시 화면에 고정되어 언제든 내용을 확인하고 수정할 수 있습니다.

### 📊 통계 및 보기 설정

- **정렬**: `최신순`, `오래된순`, `카테고리순`, `기한임박순` 등으로 리스트를 정렬할 수 있습니다.
- **드래그 앤 드롭 순서 변경**: '사용자 지정' 정렬 모드에서는 항목을 드래그하여 순서를 자유롭게 바꿀 수 있습니다.
- **통계 보기**: 상단의 `통계 보기` 버튼을 누르면 다음과 같은 상세 현황을 한눈에 볼 수 있습니다.
  - 전체 할 일 개수
  - 완료 / 미완료 개수
  - 마감 기한이 설정된 할 일 개수
  - 마감 기한이 지난 할 일 개수
  - **우선순위**: 앞으로 일주일 내에 마감해야 하는 할 일 개수
- **다크 모드**: 상단의 달(🌙) 아이콘을 눌러 눈이 편안한 다크 모드로 전환할 수 있습니다.
- **마감 기한 표시**: 각 할 일 항목에는 마감일까지 남은 일수(예: `3일 남음`, `오늘 마감!`)가 표시되며, 기한이 임박하거나 지났을 경우 색상으로 강조됩니다.

## 3. 데이터 저장

모든 데이터는 브라우저의 **로컬 스토리지(LocalStorage)**에 자동 저장됩니다. 브라우저를 닫거나 새로고침해도 데이터는 유지됩니다.
