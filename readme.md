# ✅ Smart ToDo List

**Smart ToDo List**는 개인의 일정 관리와 생산성 향상을 위해 개발된 웹 애플리케이션입니다.
직관적인 UI와 강력한 기능을 통해 할 일을 체계적으로 관리할 수 있습니다.

## 🚀 주요 기능

- **할 일 관리**: 추가, 수정(인라인), 삭제, 완료 처리
- **📅 캘린더 연동**: 마감일 확인 및 드래그 앤 드롭으로 기한 변경
- **📊 통계 대시보드**: 우선순위 및 진행 상황 시각화
- **🎨 다크 모드**: 눈이 편안한 테마 지원
- **👆 드래그 앤 드롭**: 자유로운 순서 변경
- **💾 자동 저장**: 별도 로그인 없이 브라우저에 데이터 저장 (LocalStorage)

## 📦 설치 및 실행

이 프로젝트는 별도의 서버나 설치 과정이 필요하지 않습니다.

1. 저장소를 클론하거나 다운로드합니다.
   ```bash
   git clone https://github.com/seungminbluebox/TodoList.git
   ```
2. `index.html` 파일을 웹 브라우저(Chrome, Edge 등)에서 실행합니다.

## 💻 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: LocalStorage API
